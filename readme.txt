
 About
-------

"Picture Puzzle" is hello-world project developed with pure JavaScript
(without any framework).


 How To
--------

To start "Picture Puzzle" you should open "main.htm" with Mozilla Firefox
(the project wasn't tested in other browsers).

To begin game click the button in the lower right corner.

To swap two neighbour puzzle elements drag the first element towards
the second one with button pressed mouse.

When the puzzle is solved the artist name and picture name will appear
instead of timer at the bottom of window.


 Adding Pictures
-----------------

To add picture to the puzzle you should:

1. Copy picture JPEG-file to the folder "img";
2. Add picture description string to the file "imgbase.js".
